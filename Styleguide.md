# Afficho — Style Guide

> **This is a reference copy.** The canonical style guide lives in
> [`afficho-brand/Styleguide.md`](../afficho-brand/Styleguide.md).
>
> If you need to update brand colors, typography, spacing, or assets,
> edit the canonical file and propagate changes to consuming projects.

The SVGs in `web/static/` (icon.svg, logo.svg) are copies of the originals
in `afficho-brand/assets/`. They are embedded into the Go binary at build
time. When updating these, copy the latest versions from `afficho-brand`.
